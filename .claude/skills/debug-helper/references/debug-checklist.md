# デバッグチェックリスト

## 基本確認

- [ ] エラーメッセージを正確に読んだか
- [ ] スタックトレースの発生箇所を確認したか
- [ ] 最近の変更（コミット履歴）を確認したか
- [ ] 再現手順を明確にできるか

## データ確認

- [ ] 入力データは期待通りか
- [ ] null/undefined が混入していないか
- [ ] データ型は正しいか（文字列 vs 数値など）
- [ ] 配列/オブジェクトの構造は正しいか

## 環境確認

- [ ] 環境変数は正しいか
- [ ] 依存ライブラリのバージョンは正しいか
- [ ] キャッシュが古くないか（npm, ブラウザ）
- [ ] ローカルと本番で差異はないか

## 再現確認

- [ ] 最小限の再現コードを作成できるか
- [ ] 特定の条件でのみ発生するか
- [ ] 頻度は（常に/時々/まれに）

## よくある原因

### 型の不一致

```typescript
// 文字列と数値の比較
'10' == 10   // true（型変換あり）
'10' === 10  // false（厳密比較）
```

### 非同期処理

```typescript
// await 漏れ
const data = fetchData();  // Promise が返る
const data = await fetchData();  // 値が返る
```

### スコープ

```typescript
// クロージャの落とし穴
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100);
}
// 出力: 3, 3, 3（let を使えば 0, 1, 2）
```

### 参照と値

```typescript
// オブジェクトの参照
const a = { x: 1 };
const b = a;
b.x = 2;
console.log(a.x);  // 2（参照が共有されている）
```
