# よくあるボトルネックと対処法

## データベース

| ボトルネック | 症状 | 対処 |
| ------------ | ---- | ---- |
| N+1 クエリ | クエリ数が多い | JOIN, Eager Loading, DataLoader |
| フルスキャン | クエリが遅い | インデックス追加、クエリ最適化 |
| ロック競合 | 待ち時間が長い | トランザクション短縮、楽観ロック |
| コネクション枯渇 | 接続エラー | プールサイズ調整、コネクション管理 |

## アプリケーション

| ボトルネック | 症状 | 対処 |
| ------------ | ---- | ---- |
| 同期処理 | レスポンス遅延 | 非同期化、キュー処理 |
| メモリリーク | メモリ増加 | 参照の解放、プロファイリング |
| CPU バウンド | CPU 高負荷 | アルゴリズム最適化、並列化 |
| I/O バウンド | 待ち時間 | 非同期 I/O、バッファリング |

## フロントエンド

| ボトルネック | 症状 | 対処 |
| ------------ | ---- | ---- |
| 大きなバンドル | 初期読み込み遅い | コード分割、Tree Shaking |
| 無駄な再レンダリング | UI がもたつく | メモ化（React.memo, useMemo） |
| 画像最適化不足 | 読み込み遅い | 圧縮、WebP、lazy loading |
| レイアウトスラッシング | ガタつき | バッチ DOM 操作 |

## ネットワーク

| ボトルネック | 症状 | 対処 |
| ------------ | ---- | ---- |
| 多すぎるリクエスト | 遅延 | バッチ化、HTTP/2 |
| ペイロードが大きい | 転送遅い | 圧縮、ページネーション |
| キャッシュ未活用 | 重複リクエスト | CDN、ブラウザキャッシュ |
